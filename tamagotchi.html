Tamagotchi

Qu'est-ce que c'est ?

Le projet du tamagotchi est un jeu en C qui a été réalisé lors de ma deuxième année à l'ESGI.
L'objectif du jeu ? Prendre soin de sa créature en la nourrissant et la soignant. Pour cela, il est possible de gagner des pièces au travers de mini jeu afin de pouvoir acheter le nécessaire et même quelques tenues.
Pour réaliser ce programme j'avais toutefois quelques contraintes, comme l'utilisation d'une bibliothèque graphique, l'utilisation d'une CURL et démontrer ma maîtrise des pointeurs et des structures en C.

Tamagotchi

Programmation

Ce jeu a été réalisé en langage C. Le but premier était de montrer sa maîtrise du langage, mais notament des pointeurs et des structures.


Organisation

Afin d'organiser au mieux le projet et de le rendre plus facile d'utilisation travaillant sur linux, j'ai décidé de créer un MakeFile afin de permettre une compilation rapide.
J'ai réalisé un fichier .sh afin de pouvoir en partie y mettre les prototypes de fonctions les plus importants.
J'ai organisé mon code dans différents fichiers en fonction des features.


CURL - API : OpenWeather

Afin de mettre en place l'utilisation d'une CURL dans mon jeu, j'ai réalisé un décors qui se met à jour en fonction de la météo qu'il fait lorsqu'une personne lance le jeu.
Pour cela, je fais appel à l'API "OpenWheather", qui permet de récolter en temps réel différentes données météorologiques pour n'importe quel endroit sur Terre.
Les données récoltées sont retranscrites au format json. Il m'a donc fallut parcourir ce fichier en C afin d'accéder aux informations qui m'étaient nécessaires pour afficher la bonne météo.


Bibliothèque graphique : GTK

L'une de mes contraintes pour ce projet étaient de réaliser une application avec une interface graphique.
Pour cela j'ai découvert une bibliothèque graphique qui est utilisé en C qui se nomme "GTK". Cette bibliothèque est utilisée pour créer des applications et/ou des jeux sur Linux.
Pour créer les interfaces j'ai utilisé l'application "Glade" qui permet de modéliser des interfaces utilisables par la bibliothèque GTK.


Fichier de configuration

Afin de rendre le jeu plus modulable, j'ai créé un fichier de configuration dans lequel les utilisateurs peuvent modifier certains paramètres comme le lieu où est prise la météo, les niveaux de barres de faim, d'hygiène et de soin, le niveau de difficulté pour le mini jeux ...
