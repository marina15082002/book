Formulaire

Qu'est-ce que c'est ?

Dans le cadre de mon alternance chez Sereneo, j'ai été ammené à travailler sur des applications web JavaScript comme par exemple la mise en place d'un formulaire permettant de créer et modifier un fichier JSON. Celui-ci permet à partir d'une définition JSON créer automatiquement un formulaire afin qu'une personne ne connaissant pas le JSON puisse remplir les informations. En validant le formulaire, cela génère automatiquement un fichier JSON correspondant qui est ensuite envoyé au serveur.

Formulaire

Programmation

Organisation de l'automatisation

Afin de pouvoir automatiser le code pour qu'il puisse créer un formulaire à partir de n'importe quel JSON, j'ai défini des règles dans un fichier JSON de définition sur lequel je peux m'appuyer et définir les différents éléments de bases.
Le programme peut ainsi reconnaître quand mettre en place un titre, un champ input, un champ checkbox, un tableau ou encore un select même si le nom des objets du JSON sont différents et font différentes tailles.
Pour les tableaux, j'ai ajouté une fonctionnalité qui permet d'ajouter ou de supprimer des valeurs d'un tableau et de masquer cette fonctionnalité lorsqu'il n'y a qu'un seul élément dans celui-ci.

Drag and Drop

Pour des raisons métier, j'ai ajouté une optionnalité de drag and drop sur certains objets. Pour se faire, je n'ai pas utilisé de bibliothèque existante et j'ai créé cette fonctionnalité moi-même à l'aide du JavaScript et du JQuery. Cela permet ainsi de pouvoir modifier la place d'objet ou d'éléments d'un tableau dans le JSON final. De plus, toujours dans la logique métier de notre client, une mise en place de priorité à été nécessaire lors de la construction de ce drag and drop. C'est-à-dire que certains éléments ne peuvent pas être placé avant ou après certains objets.

Documentation

Enfin, lors de mon travail sur ce projet, j'ai réalisé une documentation de l'utilisation de l'application web mais aussi une documentation ressencent le fonctionnement de mon code afin que si un jour l'application ait besoin d'une mise à jour, un autre développeur puisse reprendre facielement le travail.
