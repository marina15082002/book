Scanner de code barre

Qu'est-ce que c'est ?

L'association fictive pour laquelle j'ai créé le projet souhaite pouvoir référencer les aliments récupérés et obtenir une fiche d'information sur chacun. Pour cela j'ai créé une application en C qui permet de saisir le numéro de code barre d'un produit, puis de récupérer à l'aide d'un appel API OpenFoodFact un fichier JSON contenant toutes ces informations. Pour ne pas garder un fichier trop long et illisible pour l'utilisateur, j'ai étudié le fichier puis j'ai ajouté une fonctionnalité en C qui permet de créer un nouveau fichier JSON dans lequel je ne mets que les informations nécessaires au métier. Enfin, je transferts le fichier sur le site web afin qu'il puisse être téléchargé directement sur l'interface internet.
